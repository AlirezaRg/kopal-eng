New File: seismic_inpainting_pipeline.py
----------------------------------------------------------------

A complete CLI pipeline (no GUI) with these steps:

   1. Data center = MDA_GAN folder (data + weights)
   2. Load .npy or .segy seismic volume
   3. Define damaged region (rectangle)
   4. Reconstruct with pretrained MDA GAN model
   5. Display 4 panels: Original / Damaged / Reconstructed / Error Map
   6. Calculate MSE and PSNR

   Settings at the top of the file:
   INPUT_FILE          = file path (None = auto first file in data/)
   DAMAGE_INLINE_START = start of damaged region
   DAMAGE_INLINE_END   = end of damaged region
   DISPLAY_CROSSLINE   = which crossline to display

----------------------------------------------------------------
Technical Notes
----------------------------------------------------------------

- Python 3.11 required (PyTorch does not support Python 3.13+)
- Packages installed: torch, torchvision, matplotlib, PyQt5, numpy
- Model: MDA_GAN.pt (TorchScript, float32 on CPU)
- Default dataset: F3_salt.npy (128 x 384 x 384)
